## � 使用说明
1. 将配置文件放入 `Counter-Strike Global Offensive\game\csgo\cfg` 目录
2. 启动游戏后通过控制台执行配置
3. 建议配合离线练习模式使用

> 💡 提示：需在本地服务器/练习模式下生效
